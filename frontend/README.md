# 🥬 Veg Vibe Frontend

A modern React + Vite frontend for the Veg Vibe recipe recommendation engine.

## Quick Start

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

## Features

- 🎨 Modern, responsive UI
- ⚡ Fast with Vite
- 🔄 Real-time recipe recommendations
- ❤️ Favorites system with localStorage
- 📱 Mobile-friendly

## Environment Variables

Create a `.env` file based on `.env.example`:

```
VITE_API_URL=http://localhost:8000
```

## Project Structure

```
src/
├── components/        # Reusable React components
├── pages/            # Page layouts
├── services/         # API client
├── styles/           # CSS modules
├── App.jsx           # Main app component
└── main.jsx          # Entry point
```
